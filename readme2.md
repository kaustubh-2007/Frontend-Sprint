# 🚀 AI Roast My Life 2.0 — Reality Check Engine

> *“Not just a weather app… it’s your brutally honest life analyst.”* 😈

---

## 🌟 Overview

**AI Roast My Life 2.0** is a futuristic AI-powered web application that transforms real-time weather data and personal inputs into **humorous, insightful, and predictive life analysis**.

Instead of simply displaying weather, the app:

* Roasts your life 😈
* Predicts your day 🧠
* Simulates your future 🔮
* Gives actionable advice 💡

All powered by **Groq LLM** and real-time APIs.

---

## 🧠 Core Idea

This project implements a **“Data → Intelligence” pipeline**:

1. 🌐 Fetch real-time weather data
2. 🧾 Combine with user inputs (mood, sleep, plans)
3. 🤖 Process using AI (Groq LLM)
4. 🎯 Deliver intelligent, structured insights

## ⚙️ How It Works

### 1️⃣ User Input

User provides:

* City
* Mood
* Sleep hours
* Daily plans

---

### 2️⃣ Weather Fetching

* City → Latitude/Longitude (Geocoding API)
* Fetch real-time weather data

---

### 3️⃣ AI Processing

* Combine weather + user data
* Send structured prompt to Groq LLM
* Receive intelligent JSON output

---

### 4️⃣ Rendering

* Display results in interactive UI cards
* Visualize predictions, score, and future scenarios
